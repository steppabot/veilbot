Post Behind the Veil

Drop a message that’s fully anonymous to the server. No one sees your name or avatar—just the golden veil. Mods can set channels, limits, and rules to keep it fun.

Screenshot: the gold veil card your server will see.

Guess Who Wrote It

Every veil gets up to 3 guesses, one per user. Guesses are private (ephemeral) so nobody knows who you picked. Spend Veil Coins to take a shot—right or wrong, the mystery stays spicy.

Unveil the Author

Nail the guess and the veil drops: the card flips to black, shows the author’s display name, and fades in their profile picture. Buttons lock to prevent more guesses.

Track Your Wins

Check your Veil Coins and how many messages you’ve unveiled with a quick command. Keep score with friends, or flex your streak.

Server Leaderboard

Who’s the best sleuth? The leaderboard ranks top unveilers so you can crown a champion (and stoke a little friendly rivalry).

Keep the Game Going

Running low on coins? Pop open the Veil Store and top up instantly. More coins = more guesses = more chaos.

Safety First

Server admins can remove any veil that violates ToS or server rules. Playful anons, real moderation.

Command Reference

/veil <message>

Send a message behind a veil to your linked Veil channel (anonymous).

/user [@user]

View Veil stats (coins and messages unveiled) for you or another user.

/store

Open the Veil Coin store to top up instantly.

/leaderboard

See the top unveilers in this server.
Available on Premium & Elite.

/setup

Drop a setup message anywhere the bot can post.

/configure

Link the Veil channel and adjust settings.
Elite: enable Admin Logs here.

/upgrade

View your current plan and upgrade the server tier.

/remove

Remove a veil that violates ToS or server rules.

/info

Manage server plan and view bot info.
